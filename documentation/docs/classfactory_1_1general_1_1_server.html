Detailed Description

This class is critical to the integration of GUI classes, agents, etc. It uses a protocol established with the Message class. It contains instances of agents and calls their methods appropriately based on received messages from clients. So, this class runs a loop that constantly checks for new clients and creates a new thread for each.
