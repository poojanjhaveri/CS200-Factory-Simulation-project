Detailed Description

Verifies signals from server and take an action.

The signal from server should be checked in class 'LaneManagerVerifySignalsFromServer' Here, the signals get checked again and take an action to Graphics Panel or GUI Panel.

Verification and Action Takes.

The nest number is verified at the first line. "Switch Up" : On graphics panel, the nest goes up. On GUI Panel, the switch turns 'up'. "Switch Down" : On graphics panel, the nest goes down. On GUI Panel, the switch turns 'down' "Robot Takes One" : One part on nest disappears.
