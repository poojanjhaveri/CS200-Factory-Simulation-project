Message Checker ( From Server )

This function clarifies where the signal should go. The way it does it to check a specific letter inside the 'message' String. "&Camera&" : Message with cameras ( Signal : camera number + "&Camera&" ) "&Feeder&" : Message with feeders "&Lane&" : Message with lanes "&Nest&" : Message with nests
