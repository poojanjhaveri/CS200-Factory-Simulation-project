Detailed Description

Server Setups ( In V0, it is a platform )

This class sets up the server. In V0, there is no server requirements. However instead, this creates instances of two classes like below. class LaneManagerSetNetwork : Verifies signals from server class LaneManagerSendSignalsToServer : Sends signals to server
