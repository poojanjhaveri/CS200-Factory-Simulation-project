Part On Lane and Nest.

This class has information of parts on lane and nest. Each part has its image and location of x and y.

void factory.factory200.laneManager.LaneManagerPart.partMove ( )
inline

Part Movements.

The default limit is 0 for all parts. If the nest is full, the limit of all parts on lane is changed in order that they line up on the lane. If the current X location is larger than the limit, the part does not move.
