abstract boolean agent.Agent.pickAndExecuteAnAction ( )
protectedpure virtual

Agents must implement this scheduler to perform any actions appropriate for the current state. Will be called whenever a state change has occurred, and will be called repeated as long as it returns true.

Returns
true iff some action was executed that might have changed the state.
