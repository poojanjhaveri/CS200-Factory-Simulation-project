Detailed Description

takes care of movement of gantry robot, bins, purge station, feeders.

Gantry Robot Manager takes care of movement of gantry robot, bins, purge station, feeders. Inner classes are GantryRobot, GUIGantryManager,GraphicGantryPanel, GantryState, Feeder and Bins GUIGantryManager extends a JPanel with JButtons, JMenu and JLabels GraphicGantryPanel extends a JPanel which shows animation of every process before feeders GantryState stores data needed to do animation for bins, feeders,robot GantryRobot handles the movement of gantry robot
