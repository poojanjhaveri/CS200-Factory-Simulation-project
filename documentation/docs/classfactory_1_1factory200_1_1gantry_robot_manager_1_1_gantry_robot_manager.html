Detailed Description

GantryRobotManger takes care of movement of gantry robot, bins, purge station, feeders.

Gantry Robot Manager takes care of movement of gantry robot, bins, purge station and feeders. Instances in this manager are GUIGantryRobot, GRMGraphicPanel, GUIBin, GUIFeeder, PurgeStation GUIGantryManager extends JPanel with JButtons, JMenu and JLabels to do non-normative condition in V2. GraphicGantryPanel extends Manager which shows animation of every process: 1.Gantry robot move to bin(n) 2.Gantry robot picks up the bin 3.Gantry robot move the bin to feeder told by agents 4.Gantry robot dump the parts in the bin into feeder 5.Gantry robot purge the bin APIs are: -PickUpBin(Integer binNumber) -doSupplyPart(Integer binNumber, Integer feederNumber)///<use moveTo functions in GUIGantryRobot class) -doSupply
