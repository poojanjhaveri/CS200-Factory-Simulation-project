Detailed Description

Verifies signals from server and take an action.

The signal from server should be checked in class 'LaneManagerVerifySignalsFromServer' Here, the signals get checked again and take an action to Graphics Panel or GUI Panel.

Verification and Action Takes.

The lane number is verified at the first line. "Switch On" : On graphics panel, the lane runs. On GUI Panel, the switch turns 'on'. "Switch Off" : On graphics panel, the lane does not run. On GUI Panel, the switch turns 'off' "Amplitude Weak" : On graphics panel, the lane vibrates weakly. On GUI Panel, the amplitude slider turns 'weak' "Amplitude Normal" : On graphics panel, the lane vibrates normally. On GUI Panel, the amplitude slider turns 'normal' "Amplitude Strong" : On graphics panel, the lane vibrates strongly. On GUI Panel, the amplitude slider turns 'strong' "MOVE" : On graphics panel, the lane moves. This signal comes from timer in server.

message: message from server
