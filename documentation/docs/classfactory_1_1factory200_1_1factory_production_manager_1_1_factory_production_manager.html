Detailed Description

selects active kit production routines, how many of each kit to produce, and also will control non-normative scenario simulation

Factory Production Manager selects active kit production routines, how many of each kit to produce, and also will control non-normative scenario simulation. This client also displays graphics representing the entire factory simulation. Referenced classes are KitRobot, PartsRobot, GantryRobot, Nest, Lane, Bin, and CameraHandler Inner Class FPMGUI extends JPanel and is an instance of the Factory Management GUI Inner Class FPMGraphics extends Jpanel and is an instance of the entire Factory Graphics display InnerClass NonNormative extends JPanel and is an instance of the FPM Non-Normative Control Panel
